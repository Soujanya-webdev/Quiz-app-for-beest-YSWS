# BrainBuzz Quiz

## What this is

A browser-based quiz app where users can either create their own quizzes manually or play pre-generated quizzes based on subject, class, and difficulty.Everything runs fully in the browser with no backend.

## Why I built it

I wanted to build something flexible where users can both *create questions* and *play dynamically generated quizzes* in the same system.This was also built to practice DOM-based state management and UI flow handling in vanilla JavaScript.

## Features

* Create custom quizzes with your own questions and options
* Play generated quizzes by selecting subject, class, and difficulty
* Score tracking with live updates
* Multi-screen navigation (home, builder, quiz, result)
* Fully responsive UI for mobile and desktop

## How it works

The app uses JavaScript to manage quiz state (questions, score, progress). Screens are swapped using simple DOM-based view switching. Custom and generated quiz flows both feed into a shared quiz engine.
